# CSP451 CheckPoint1 - Vishal Sonar

## Project Title and Description
**Project Title:** CSP451 CheckPoint1 - Git & GitHub Basics

**Description:**  
This project demonstrates the fundamentals of version control using Git and GitHub. It includes a local repository setup, multiple meaningful commits, branching workflows and pull request management. The project contains a simple HTML, CSS and JavaScript website along with an About page that displays personal information and links.

## Contributing

This project is part of an academic assignment. Contributions are not required, but suggestions and improvements are welcome.
Fork the repository to make changes in your own branch.

Commit changes using proper commit message format:

type: description


Commit types:

feat — new features

fix — bug fixes

docs — documentation updates

style — formatting/styling

refactor — code restructuring

Submit Pull Requests for review before merging changes to main.

Test changes locally before committing.
